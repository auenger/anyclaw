# Calculator Skill

Perform mathematical calculations.

## Usage

```
calc <expression>
```

## Parameters

- `expression`: Mathematical expression to evaluate

## Supported Operations

- Basic: `+`, `-`, `*`, `/`, `%`
- Functions: `abs`, `round`, `min`, `max`, `sum`, `pow`, `sqrt`, `sin`, `cos`, `tan`, `log`, `log10`, `exp`
- Constants: `pi`, `e`

## Examples

```bash
# Basic arithmetic
calc "2 + 3 * 4"

# Using functions
calc "sqrt(16)"

# Using constants
calc "pi * 2"

# Complex expression
calc "sin(pi/4) + cos(pi/4)"
```
