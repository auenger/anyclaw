# Cron 定时任务

## 功能说明

使用 cron 技能可以创建定时任务和提醒，支持三种调度方式：

1. **间隔执行** - 每隔固定时间执行
2. **定时执行** - 指定具体时间执行（一次性）
3. **Cron 表达式** - 使用标准 cron 表达式（高级）

## 支持的操作

### 添加任务 (add)

创建一个新的定时任务。

**参数**:
- `action`: "add"
- `message`: 提醒消息内容（必需）
- `every_seconds`: 间隔秒数（循环任务）
- `at`: ISO 格式时间（一次性任务）
- `cron_expr`: Cron 表达式（高级调度）
- `tz`: 时区（配合 cron_expr 使用）

**示例请求**:

```
# 每小时提醒喝水
"帮我设置一个每小时提醒喝水的任务"

# 明天上午 9 点提醒
"明天上午 9 点提醒我开会"

# 每天早上 9 点提醒
"每天早上 9 点提醒我晨会"
```

### 列出任务 (list)

显示所有已创建的定时任务。

**参数**:
- `action`: "list"

**示例请求**:
```
"显示我的所有定时任务"
"列出所有提醒"
```

### 删除任务 (remove)

删除指定的定时任务。

**参数**:
- `action`: "remove"
- `job_id`: 任务 ID（从 list 命令获取）

**示例请求**:
```
"删除任务 xxxxxxxx"
"取消喝水提醒"
```

## 使用示例

### 间隔提醒

```
用户: 每隔 30 分钟提醒我休息一下
助手: [调用 cron 工具]
已创建任务 "每 30 分钟提醒我休息一下" (id: abc123)
下次执行时间: 2026-03-22 14:30:00
```

### 一次性提醒

```
用户: 明天下午 3 点提醒我参加面试
助手: [调用 cron 工具]
已创建一次性任务 "参加面试" (id: def456)
执行时间: 2026-03-23 15:00:00
```

### Cron 表达式

```
用户: 每个工作日上午 9 点提醒我写日报
助手: [调用 cron 工具]
已创建任务 "写日报" (id: ghi789)
使用 cron 表达式: 0 9 * * 1-5
```

## 注意事项

1. **任务持久化**: 定时任务存储在 `.anyclaw/cron_jobs.json`，重启后仍然有效
2. **最小间隔**: 建议最小间隔为 60 秒，避免过于频繁的提醒
3. **时区支持**: 使用 `tz` 参数指定时区，如 `Asia/Shanghai`
4. **任务数量**: 建议同时运行的任务不超过 10 个

## 技术细节

- 使用 CronService 管理任务调度
- 支持 asyncio 异步执行
- 自动计算下次执行时间
- 支持任务状态跟踪（成功/失败/跳过）
