# Bootstrap 说明

此文件说明 workspace 的引导文件系统。

## 文件说明

| 文件 | 用途 | 何时使用 |
|------|------|----------|
| `SOUL.md` | Agent 人格和价值观 | 定义 Agent 的性格和行为准则 |
| `USER.md` | 用户档案 | 存储用户偏好和背景信息 |
| `AGENTS.md` | Agent 指令 | 定义 Agent 的行为规则和工具使用指南 |
| `TOOLS.md` | 工具说明 | 记录工具使用的约束和最佳实践 |
| `IDENTITY.md` | Agent 身份 | 定义 Agent 的名称、头像等元信息 |
| `HEARTBEAT.md` | 心跳任务 | 定义定期执行的任务 |

## 记忆系统

| 文件 | 用途 |
|------|------|
| `memory/MEMORY.md` | 长期记忆，存储重要信息 |
| `memory/HISTORY.md` | 对话历史摘要 |

## 技能系统

自定义技能放在 `skills/` 目录下：

```
skills/
└── my-skill/
    └── SKILL.md    # 技能定义文件
```

## 最佳实践

1. **保持简洁**：引导文件不宜过长，避免占用过多 token
2. **定期更新**：随着使用调整文件内容
3. **记忆管理**：重要信息写入 MEMORY.md，而不是对话历史
4. **技能复用**：常用操作封装成技能
