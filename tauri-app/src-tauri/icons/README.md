# Icons

This directory should contain the following icon files:

- tray.png - System tray icon (16x16 or 32x32)
- 32x32.png - 32x32 icon
- 128x128.png - 128x128 icon
- 128x128@2x.png - 256x256 icon
- icon.icns - macOS icon bundle
- icon.ico - Windows icon file

For now, you can use any placeholder PNG files.
Replace them with proper icons before building the app.
