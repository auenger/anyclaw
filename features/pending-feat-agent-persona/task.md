## 关键决策

### Q1: 人设文件最大字符数？

**选项**：
1. 10000 字符（默认）
2. 20000 字符
3. 可配置

**决策**: 可配置，默认 10000 字符。

### Q2: 是否支持 YAML frontmatter？

**选项**：
1. 支持（类似 OpenClaw）
2. 不支持，纯 Markdown
3. 可选

**决策**: 支持，用于元数据（如 `read_when`）。

### Q3: 人设更新时机？

**选项**：
1. 每次会话开始
2. 实时检测文件变更
3. 手动刷新

**决策**: 每次会话开始加载，支持 `/refresh` 手动刷新。
