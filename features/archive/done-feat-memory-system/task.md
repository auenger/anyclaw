## 关键决策

### Q1: 每日日志保留多少天？

**选项**：
1. 7 天
2. 30 天
3. 无限制（手动清理）

**决策**: 无限制，提供清理命令。

### Q2: 记忆更新是否需要用户确认？

**选项**：
1. 始终需要确认
2. 可配置（默认需要）
3. 自动更新

**决策**: 可配置，默认需要确认。

### Q3: 搜索结果如何排序？

**选项**：
1. 按日期倒序
2. 按相关性
3. 按来源类型

**决策**: 按日期倒序，长期记忆优先。
